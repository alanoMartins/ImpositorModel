# Impositor Model

This is a POC and educational propose project. This code execute a gaussian mixture classificator with Iris Dataset.

### How execute

  . Download dependecies with pip:
    ``` pip install -r requirements.txt ```
    
  . Run main file:
  ```	 python main.py ```
